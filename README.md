# Barangay Health Center Patient Record and Appointment Management System

## Description

A web application that helps barangay health centers manage patients, appointments, healthcare services, and consultation records.

Statement of the Problem

Many barangay health centers still rely on paper-based records and manual appointment scheduling. This traditional method often results in misplaced patient records, slow retrieval of information, long waiting times, and scheduling conflicts. These issues reduce the efficiency of healthcare workers and affect the quality of service provided to patients.

This study aims to develop a Barangay Health Center Patient Record and Appointment Management System to address these challenges. Specifically, the study seeks to answer the following questions:

1. How can a computerized system improve the management and storage of patient records in the barangay health center?
2. How can the system provide an efficient appointment scheduling process to minimize waiting time and scheduling conflicts?
3. How can the system enable healthcare workers to quickly retrieve and update patient medical records?
4. How can the system improve the accuracy, security, and organization of patient information?
5. How can the system generate reports that support efficient monitoring and decision-making in the barangay health center?

CRUD Meaning

* C – Create: Add new data (e.g., register a patient).
* R – Read: View or search existing data.
* U – Update: Modify existing information.
* D – Delete: Remove data that is no longer needed or was entered incorrectly (usually restricted to administrators).

These CRUD operations form the core functionality of the system and allow authorized health center staff to efficiently manage patient information and appointments.
